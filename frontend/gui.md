# GUI

The **graphical user interface** is the frontend of the application. It is built with **Next.js/React** and is used to manage the application and user content (categories, questions, tests, campaigns).

It supports creating, editing, deleting, sharing, and managing user content, and is the main way users interact with the app.

## Usage

The main app is available at [Quizdo.eu](https://quizdo.eu).
