# Settings Component

## Overview

The Settings component manages user preferences, application configuration, and system-wide settings. It handles user-specific settings like language preferences, theme selection, and default subjects, as well as system-level configurations that affect the entire application.

## Purpose

- **User Preferences**: Store and manage individual user settings
- **Localization**: Handle language and regional preferences
- **UI Customization**: Manage theme and interface preferences
- **Default Configuration**: Set up default subjects and categories
- **System Configuration**: Manage application-wide settings
- **Personalization**: Customize user experience and content discovery

## Core Settings Fields

- **Locale**: User's preferred language and region (e.g., "cs-CZ", "en-GB")
- **Theme**: UI theme preference (light/dark mode)
- **Subjects**: Array of default subject category IDs
