# Services

This directory contains information about the services of the application.

`Services` are the microservices of the application. Or organizational units of the application.

- [API Content](api_content.md) is the main application API that is used by the frontend, API consumers and the backend. Shouldnt manage any user information or logging. It's used to manage the content of the application only.
- [API Live Testing](api_livetesting.md) is the API that is used by apps for live testing, managing responses and results.
- [api](api.md)

- [Generator](generator.md) is open source part of the application that is used by backend to generate the content based on given parameters.
- [GUI](gui.md) is the graphical user interface of the application that is used to manage their own content and generate exams from it.
