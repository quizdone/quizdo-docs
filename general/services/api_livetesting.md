# Live testing

## Description

Live testing is a tool to use generated exams and single questions and gather results in real time using WebSocket.

It also provides way of interacting with campaigns and advancing to the next exam based on the results.

It's separated service with it's own endpoints and API keys generated inside the application separately.

Uses WebSocket to gather results in real time and can be used in apps to test students in real time, evaluate in the app, watch for cheating etc.
