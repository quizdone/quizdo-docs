# Content API

## Description

This part of the app is accessible via API and performs CRUD (Create, Read, Update, Delete) operations on the user content of the application.

So far used by NextJS app to provide user interface for managing and displaying examples, tests, campaigns.

The app can also share the content with other users, read and copy questions (like social networks) and use it in their own projects using API.

It will serve as simple social network for sharing and using content between users.
