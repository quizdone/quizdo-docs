# Generator

## Description

Generator is a (command line) tool to generate tasks (questions) from a given data.
It's written in Go and can be used as a library or as a command line tool.

The generator is used in the CRUD API to check and generate tasks (questions) from a given data.

> **This app lives in it's own repository**.
>
> Find more up to date information in the [generator](https://gitlab.com/pisemkomat/generator) repository documentation.
